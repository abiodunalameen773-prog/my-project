# Royal Gate Hotel Website

## Quick Deploy to GitHub Pages (Free)

1. Create account at github.com
2. Create new repository named "royal-gate-hotel"
3. Upload all your files (index.html, style.css, class.js)
4. Go to Settings > Pages
5. Select "Deploy from a branch" > "main"
6. Your site will be live at: https://yourusername.github.io/royal-gate-hotel

## Alternative: Use ngrok for instant sharing

1. Download ngrok from https://ngrok.com/download
2. Run: `http-server -p 8080`
3. Run: `ngrok http 8080`
4. Share the https://xxx.ngrok.io link with anyone!

## Files in this project:
